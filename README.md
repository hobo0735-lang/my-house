# 屏幕彩边边框工具

这是一个基于 Python 的桌面美化工具，可以在屏幕边缘显示自定义的动态边框，支持流光特效，非常适合作为直播、游戏或演示时的视觉辅助。

## 主要功能
* **置顶显示**：支持全屏透明置顶，不遮挡鼠标操作。
* **极光流光模式**：🌈 可开启 RGB 动态流光效果。
* **高度可定制**：支持调整边框粗细、字体大小、文字内容及颜色。
* **快捷唤醒**：内置全局快捷键（默认 F9），随时一键切换显示/隐藏。

## 如何使用

### 方式 1：直接运行成品 (推荐)
1. 访问右侧的 **Releases** 页面。
2. 下载最新的 `.exe` 压缩包并解压。
3. 请确保 **`内容.json` (配置文件)** 与 `.exe` 文件位于同一目录下。
4. 双击运行 `.exe` 即可。

### 方式 2：从源码运行 (供开发者参考)
1. 确保已安装 Python 环境。
2. 安装依赖库：
   ```bash
   pip install keyboard

1. 运行主程序：

   Bash

   ```
   python src/border2_第三版.py
   ```

## 配置说明

- 程序会自动读取同目录下的 `内容.json` 文件以保存你的偏好设置。
- 如需修改快捷键或文字，可以通过程序自带的配置面板修改，点击“应用并保存”即可自动更新 JSON。

- 本项目为最终版本，不再进行功能更新。
- 运行程序时请保持 `内容.json` 文件存在，否则程序将恢复默认配置。
- 该程序运行需要一定的系统权限（用于监听快捷键）。
