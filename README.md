这是一个基于 Python + Tkinter 开发的桌面辅助工具。它可以在你的屏幕四周生成一圈丝滑的、带流光特效的彩色边框，支持自定义文字提醒、快捷键唤醒以及实时调速。
核心特性
动态流光：采用 HSV 色彩空间计算，支持 100 段平滑渐变的“极光”特效。
全局监听：使用 keyboard 库实现全局快捷键唤醒（默认 F9），即便在游戏或全屏应用中也能一键切换。
智能记忆：所有设置（颜色、粗细、文字、速度）均自动保存至 t内容.json，下次启动无需重设。
高度自定义：
  1. 屏幕四个角可独立设置提醒文字
  2. 实时滑动条调节流光速度。
  3. 自定义边框粗细与字体大小。

windows版本
1. 环境准备
确保你的电脑已安装 Python、

2. 安装依赖
在终端中运行以下命令安装必要的库：
pip install keyboard

3. 使用：打开代码，复制到本地项目，并在项目目录建一个json文件，命名为t内容，然后再运行代码，在弹出的界面中选好属性再点击应用设置并固定到文件即可

4. 开关快捷键：f9



mac版本
1安装依赖
在终端中运行以下命令安装必要的库：
pip install keyboard

2运行代码并授权（Mac 用户最关键的一步）
macOS 对隐私保护非常严格，默认是不允许任何程序监听键盘按键的。

启动程序：在你的编辑器（如 VS Code）或终端中运行刚才那段代码。

触发授权：你会发现程序运行了，但按快捷键没反应。

前往设置：

打开 系统设置 (System Settings) -> 隐私与安全性 (Privacy & Security) -> 辅助功能 (Accessibility)。

点击下方的 + 号，找到并添加你正在使用的工具（比如 终端/Terminal、VS Code 或 PyCharm）。

确保开关处于开启状态。

第三步：操作指南
程序启动后会弹出一个**“设置面板”**，你可以这样操作：

修改文字：在“左上文字”、“右上文字”等输入框里填入你想看到的提醒（比如：保持专注！、多喝水）。

开启流光：勾选 “开启极光流光”，边框就会开始变色。

调节速度：拉动滑动条，可以实时看到边框变色的快慢。

应用并保存：点击 “应用并保存”。此时程序会在你的代码同级目录下生成一个 t内容.json 文件，下次打开时，你的所有设置都会自动找回来。

开启/隐藏边框：

方法 A：按下默认快捷键 Command + Shift + B。

方法 B：直接点击设置面板上的 “开关边框” 按钮。

如果成功了，没有保存文件，第一次运行是默认的边框，设置完毕点击应用并保存按钮，会生成一个json文件，下次打开，就固定效果了
