这是一个基于 Python + Tkinter 开发的桌面辅助工具。它可以在你的屏幕四周生成一圈丝滑的、带流光特效的彩色边框，支持自定义文字提醒、快捷键唤醒以及实时调速。
核心特性
动态流光：采用 HSV 色彩空间计算，支持 100 段平滑渐变的“极光”特效。
全局监听：使用 keyboard 库实现全局快捷键唤醒（默认 F9），即便在游戏或全屏应用中也能一键切换。
智能记忆：所有设置（颜色、粗细、文字、速度）均自动保存至 t内容.json，下次启动无需重设。
高度自定义：
  1. 屏幕四个角可独立设置提醒文字
  2. 实时滑动条调节流光速度。
  3. 自定义边框粗细与字体大小。

详细的打包步骤：

### 🛠️ 第一步：安装打包工具

打开你的命令行终端（CMD、PowerShell 或代码编辑器里的终端），输入以下命令安装 PyInstaller：

Bash

```
pip install pyinstaller
```

### 🚀 第二步：执行打包命令

在终端中，通过 `cd` 命令切换到你 Python 代码所在的文件夹。假设你的代码文件名为 `main.py`，请输入以下命令进行打包：

Bash

```
pyinstaller -F -w --uac-admin main.py
```

**参数大解密（为什么用这些参数）：**

- **`-F` (或 `--onefile`)**：将所有依赖项和环境打包成一个独立的 `.exe` 单文件，方便分享。
- **`-w` (或 `--noconsole`)**：**非常重要！** 隐藏黑色的命令行控制台窗口。因为你是界面的程序，如果不加这个，运行时背后会一直挂着一个丑陋的黑框。
- **`--uac-admin`**：**针对你代码的特调！** 因为你用了 `keyboard` 库监听全局快捷键（F9），这在 Windows 中属于底层操作，必须拥有管理员权限。加上这个参数后，双击运行 exe 时会自动弹出“是否允许应用进行更改”的管理员授权窗口，避免快捷键失效。

*(可选)* 如果你想给你的程序加个自定义图标，可以准备一个 `.ico` 格式的图片，加上 `-i` 参数：

Bash

```
pyinstaller -F -w --uac-admin -i your_icon.ico main.py
```

### 📦 第三步：获取你的 `.exe` 文件

打包过程可能需要几十秒到几分钟，终端显示 `completed successfully.` 后即告完成。

这时，你的代码目录下会自动生成几个新文件夹。你只需要关注 **`dist`** 文件夹。 打开 `dist` 文件夹，里面那个 `main.exe` 就是你最终的成品！你可以把它复制到电脑的任何地方运行，不再需要安装 Python 环境。
